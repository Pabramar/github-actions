# GitHub Actions Course Repository

This repository contains practical examples and workflows for learning GitHub Actions.

## Structure

- **01 Starting Project/**: Basic HTML project for initial exercises.
- **04-using-actions/**: Contains a sample React application for demonstrating actions and CI workflows.
- **.github/workflows/**: YAML files with various workflow examples:
  - Building blocks
  - Workflow events
  - Workflow runners
  - Using actions
  - Event filters and activity types

## How to Use

1. Clone the repository.
2. Explore the workflow files in `.github/workflows/`.
3. Try running workflows using the GitHub Actions tab.
4. Modify or add your own workflows to experiment.
